# Project Statement

Student Performance Predictor is a Machine Learning based project developed using Python. The purpose of this project is to predict a student's final exam score based on factors such as hours studied, attendance percentage, and previous exam marks.

The system uses Linear Regression to analyze the relationship between these inputs and the final score. A simple graphical user interface built with Tkinter allows the user to enter values and instantly receive a predicted result along with a performance comment.

This project helps in understanding the practical use of Machine Learning, data handling with Pandas, model training with Scikit-learn, and GUI development with Tkinter.

The project can be useful for teachers, students, and educational institutions to identify students who may need extra attention and improvement.
